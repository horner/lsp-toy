About

This is a Language Server Protocol (LSP) implementation for the LSP-Toy language. The server communicates via WebSocket and is designed to work with VS Code and other LSP-compatible editors.

Setup

To use this language server:

  1. Install the LSP-Toy extension in VS Code
  2. The extension will automatically connect to this WebSocket server
  3. Open any .lsptoy file to activate language features
